# Tengu Homebrew Tap

Homebrew formulae for [Tengu](https://tengu.to) tools.

## Installation

```bash
brew tap tengu-apps/tap
brew install tengu-init
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `tengu-init` | Provision Tengu PaaS servers via SSH |

## Updating

```bash
brew update
brew upgrade tengu-init
```
